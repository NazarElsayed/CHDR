Description

template<typename T>
class chdr::queue< T >

Implementation of a First-In-First-Out (FIFO) container.

Manages a collection of elements, maintaining a FIFO ordering.

Template Parameters
TThe type of elements stored in the queue.
Note
This class uses polymorphic memory resources (std::pmr::memory_resource).
Follows an STL-like design and supports iterators.
