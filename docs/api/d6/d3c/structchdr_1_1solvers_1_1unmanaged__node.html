unmanaged_node< index_t >

Description

template<typename index_t>
struct chdr::solvers::unmanaged_node< index_t >

A hierarchical node in a pathfinding context, requires manual memory management.

This structure includes a pointer to its parent, allowing it to handle the hierarchical relationship between nodes.

Warning
unmanaged_node does not manage the lifetime of its parent. Careful considerations should be made to ensure it is cleaned up correctly.
Template Parameters
index_tThe type representing the identifier, derived from bnode. It must be an integral type.
