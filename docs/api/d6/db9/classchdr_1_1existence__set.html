inlineconstexpr

resize the existence_set.

This method resizes the existence_set by calling the resize method on the internal vector.

Parameters
_newSizeThe new size to resize the set to.
_newValueThe optional value to fill the new elements with. (default is false).
Note
If the new size is smaller than the current size, the elements at the end will be removed. If the new size is greater than the current size, the new elements will be filled with the specified value.
