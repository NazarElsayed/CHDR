template<typename index_t , typename scalar_t >
const neighbours_t & get_neighbours ( index_t  _id) const
inline

Retrieves the neighbours associated with a vertex in the graph.

This method fetches a constant reference to the neighbours of the vertex with the given identifier.

Parameters
_id[in] The index of the node whose neighbours are to be retrieved.
Returns
A constant reference to the neighbours associated with the specified node ID.
Warning
If the requested node does not exist within the graph, calling this function invokes undefined behaviour.
