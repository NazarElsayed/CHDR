existence_set_example.cpp

Demonstrates the usage of the existence_set class.

Demonstrates the usage of the existence_set class.This example shows how to construct an existence_set, perform insertion, check existence, and disable specific elements.

#include "existence_set.hpp"
#include <iostream>
int main() {
// Using high_performance layout (default).
chdr::existence_set<> set;
// Enable elements with hashes 1, 2, and 5.
set.push(1);
set.push(2);
set.push(5);
// Check if elements exist.
if (set.contains(1)) { // true
// ...
}
if (set.contains(3)) { // false
// ...
}
// Disable an element.
set.erase(2);
return 0;
}
