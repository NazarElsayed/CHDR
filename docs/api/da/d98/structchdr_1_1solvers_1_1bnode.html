template<typename index_t>
struct chdr::solvers::bnode< index_t >

Base class for pathfinding nodes.

bnode represents the fundamental functionality of a node in a search context, storing an index representing an identifier. It can be inherited to extend its functionality.

Remarks
The stored identifier does not need to be unique.
Template Parameters
index_tThe type of the identifier. Must be an integral type.
