Description

template<typename coord_t, typename weight_t>
class chdr::mazes::grid< coord_t, weight_t >

Represents a uniform-cost, bidirectional grid in K-dimensions.

Note
This class uses a coord_t type for grid dimensions and indexing, and a weight_t type to represent the weight or state of the nodes in the grid.
Template Parameters
coord_tType representing coordinates.
weight_tType representing node weights.
