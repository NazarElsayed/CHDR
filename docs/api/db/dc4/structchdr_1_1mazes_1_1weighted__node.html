weighted_node< weight_t >

Description

template<typename weight_t>
struct chdr::mazes::weighted_node< weight_t >

Represents a node with customisable weight.

Encapsulates a user-defined integral type.

Note
Is equivalent to its stored weight and can be reinterpreted as the weight's type.
Remarks
Stored weight may be signed or unsigned.
Warning
The maximum value of weight_t is reserved for inactive nodes.
Template Parameters
weight_tWeight type. Must be integral.
