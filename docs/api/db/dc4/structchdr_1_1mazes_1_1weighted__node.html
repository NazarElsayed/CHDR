weighted_node< weight_t >

Description

template<typename weight_t>
struct chdr::mazes::weighted_node< weight_t >

Represents a node with a customisable weight.

The weighted_node class encapsulates a weight of a user-defined integral type.

Note
The weighted_node class is equivalent to its stored weight and can be reinterpreted as the weights's type.
Remarks
The stored weight may be signed or unsigned.
Warning
For weighted_node instances, the maximum value of weight_t is reserved for inactive nodes.
Template Parameters
weight_tThe type representing weight. Must be an integral type.
