template<typename index_t>
class chdr::mazes::id_node< index_t >

Represents a node with a customisable identifier.

The id_node class encapsulates an identifier of a user-defined integral type.

Note
The id_node class is equivalent to its stored identifier and can be reinterpreted as the identifier's type.
Remarks
The stored identifier does not need to be unique.
Template Parameters
index_tThe type of the identifier. Must be an integral type.
