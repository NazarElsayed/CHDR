stack< T >

Description

template<typename T>
class chdr::stack< T >

Implementation of a First-In-Last-Out (FILO) container.

Manages a collection of elements, maintaining a FILO ordering.

Template Parameters
TType of elements stored in the stack.
Note
This class uses polymorphic memory resources (std::pmr::memory_resource).
Follows an STL-like design and supports iterators.
