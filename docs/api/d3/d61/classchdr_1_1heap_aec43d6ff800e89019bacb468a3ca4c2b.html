Checks whether the specified item exists in the heap container.

This function evaluates if a given item _item is present within the heap by determining its index and verifying its equality with the corresponding element in the container c. The function ensures that the heap is not empty before proceeding.

Parameters
_item[in] A reference to the item whose presence in the heap is to be checked.
Returns
bool True if the item exists in the heap container; otherwise, false.
