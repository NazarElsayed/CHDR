Reorganises the heap by moving a specified item downward to maintain the heap property.

This function moves the specified item downward within the heap structure to restore the heap's ordering property. It is typically used after a modification or removal operation that causes the heap to become unbalanced. The operation is efficient, utilising comparisons and optional element swaps.

Parameters
_item[in] The item to be moved downward within the heap to restore the heap property.
