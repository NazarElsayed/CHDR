template<typename T , typename Compare = std::less<T>, typename Container = std::pmr::vector<T>, size_t Kd = 2U>
T dequeue ( )
inlinenoexcept

Removes and returns the top element from the heap.

This function extracts the element at the top of the heap, ensuring the heap's internal structure remains valid after the operation. The top element is replaced with the last element in the container before being removed, and the internal order of the heap is restored using sort_down.

Returns
T The element extracted from the top of the heap.
Precondition
The heap must not be empty before calling this function.
Remarks
This function assumes the heap is not empty.
Warning
Behaviour is undefined if called on an empty heap.
