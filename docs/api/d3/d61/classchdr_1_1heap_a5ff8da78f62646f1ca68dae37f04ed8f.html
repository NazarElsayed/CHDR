void erase ( const T &  _item)
inlinenoexcept

Removes a specified item from the heap, restoring the heap property after removal.

This function erases a given item from the heap container, ensuring the heap structure's integrity is preserved. If the item to be removed is the last item, it is simply removed. Otherwise, the last element replaces the erased item and the heap property is restored through appropriate heap adjustments.

Parameters
_item[in] The item to be removed from the heap container.
