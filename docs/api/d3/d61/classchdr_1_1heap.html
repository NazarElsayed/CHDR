heap< T, Compare, Container, Kd >

Description

template<typename T, typename Compare = std::less<T>, typename Container = std::pmr::vector<T>, size_t Kd = 2U>
class chdr::heap< T, Compare, Container, Kd >

A generic heap implementation with configurable properties.

Implements a heap data structure, allowing priority-based storage and retrieval of elements. It supports custom comparator functions and uses a container that manages its items internally. The heap is optimised for performance and includes methods for managing its structure while maintaining the heap property.

Dimensionality of the heap can be customised through the template parameters.
