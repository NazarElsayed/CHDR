Description

template<template< typename params_t > typename solver_t, typename params_t>
class chdr::solvers::solver< solver_t, params_t >

A static class serving as the core entry point for the CHDR library.

The solver class provides a unified interface for instantiating, configuring, and invoking various solvers. It consolidates common utilities and functionalities required to create and manage search or optimisation instances. This class is designed for flexibility and extensibility, supporting a wide range of solvers through the use of template specialisations.

Key features include:

  • Integration with customisable parameter sets defined by params_t.
  • Specialized utility traits to enforce type and method requirements during compilation.
  • Support for user-defined solver implementations via the solver_t template parameter.
// Example usage. Replace [...] with your values.

};
