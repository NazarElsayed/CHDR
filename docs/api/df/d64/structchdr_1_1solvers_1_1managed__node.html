managed_node< index_t, derived >

Description

template<typename index_t, typename derived = void>
struct chdr::solvers::managed_node< index_t, derived >

A hierarchical node in a pathfinding context, with a clean-up mechanism.

This structure includes a pointer to its parent, allowing it to handle the hierarchical relationship between nodes. It uses a reference counting mechanism to aid with resource management.

Warning
Clean-up must be triggered manually, via the expunge function.
Template Parameters
index_tThe data type used for indexing or identifying the node.
derived(optional) derived class type for CRTP (Curiously Recurring Template Pattern) usage.
