◆ forget_one()

template<typename index_t , typename derived = void>
auto * forget_one ( )
inline

Decrements parent's successor count and checks if it can be released.

This function decrements the successor count of the parent node if one exists. If the parent's successor count reaches zero after decrementing, the parent node is marked for potential release by returning a pointer to it. If there is no parent node, nullptr is returned.

Returns
A pointer to the parent node if it can be released (has zero successors), nullptr otherwise.
Warning
The actual release/deallocation of the returned node must be handled by the caller. This function only indicates which node is eligible for release.
Precondition
The parent node's successor count must be greater than 0 before decrementing. This is checked via assertion.
