Maps a multi-dimensional index to a one-dimensional array index.

Calculates the flattened one-dimensional index corresponding to the specified multi-dimensional indices _indices, given the dimensions and sizes of the multi-dimensional space _sizes. Supports 1â€“N dimensional spaces, with explicit optimisations performed for the first four dimensions.

Parameters
[in]_indicesA coordinate representation where each element specifies the index along a particular dimension. The size of this parameter must match the number of dimensions.
[in]_sizesThe sizes of each dimension in the multi-dimensional space. Each element defines the maximum size along the corresponding dimension. The size of this parameter must match the number of dimensions.
Returns
The computed one-dimensional array index as a value of type T, where T represents the integral type used by coordinate elements.
