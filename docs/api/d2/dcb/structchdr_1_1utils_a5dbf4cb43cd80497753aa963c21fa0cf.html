Computes the absolute value of a given arithmetic type.

This function returns the absolute value of the input parameter, ensuring correct handling of signed and unsigned arithmetic types. If the input type is unsigned, the value is returned as-is. For signed types, a negative value is converted to its positive equivalent.

Parameters
[in]_valueThe arithmetic value for which the absolute value is to be computed. The type T must satisfy the arithmetic constraint, as enforced by static_assert.
Returns
The absolute value of the input parameter. If the input is unsigned or non-negative, the original value is returned. For signed negative values, the positive equivalent is returned.
