Returns the smaller of two values.

Compares two values of type T and returns the smaller of the two. The function imposes a static assertion to ensure that the type T supports the less-than operator for comparison.

Parameters
[in]_aThe first value to be compared.
[in]_bThe second value to be compared.
Returns
A reference to the smaller of the two values. If both values are equal, the reference to the first argument _a is returned.
