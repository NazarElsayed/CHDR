Retrieves the neighbours of a node within the grid.

Computes the neighbours of the specified node in the grid, either including or excluding diagonal neighbours. The neighbours allow navigation and provide information about the connectivity of the node to its surroundings.

Parameters
_idThe coordinate of the node for which the neighbours are to be retrieved.
Template Parameters
IncludeDiagonalsBoolean template parameter indicating whether to include diagonal neighbours.
  • If true, diagonal neighbours are included.
  • If false, only axis-aligned neighbours are considered.
Note
Out-of-bounds neighbours are marked inactive. Activity of a neighbour is determined by its state within the grid. The total number of neighbours depends on the dimensionality (rank) of the grid and whether diagonal neighbours are included.
Returns
An array of pairs where each pair represents:
  • A boolean indicating whether the neighbour is "active".
  • The coordinate of the neighbour (invalid if the node is inactive).
Warning
In the resulting array of pairs, if a node is inactive, its corresponding coordinate is invalid and should be used.
