struct chdr::solvers::solver< Derived, params_t >::solver_utils::is_graph< T >

A utility trait to check if a type is a graph type.

This struct determines if a given type T, when decayed, is equivalent to the graph type mazes::graph with index and scalar types supplied by parameters from params_t. This comparison utilises std::is_same and std::decay_t for precise type matching.

Note
This trait is primarily used within the solver class to ensure compatibility with graph structures in mazes, specifically verifying if the type satisfies graph type requirements for operations.
